# Hello World Random

Return one greeting randomly selected from five languages.

## Quick use

Run:

```bash
python3 scripts/hello_random.py
```

The script prints exactly one line.

## Languages included

- English: Hello, World!
- Portuguese: Olá, Mundo!
- Spanish: ¡Hola, Mundo!
- French: Bonjour, le monde !
- German: Hallo, Welt!

## Notes

- Keep output short: one greeting only.
- If a user asks to reroll, run the script again.
