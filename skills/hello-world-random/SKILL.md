# Hello World Random

Return five greetings (one per language) in random order.

## Quick use

Run:

```bash
python3 scripts/hello_random.py
```

The script prints five lines (all languages), shuffled randomly.

## Languages included

- English: Hello, World!
- Portuguese: Olá, Mundo!
- Spanish: ¡Hola, Mundo!
- French: Bonjour, le monde !
- German: Hallo, Welt!

## Notes

- Keep output short: one greeting only.
- If a user asks to reroll, run the script again.
